# 529 Savings Calculator

A Progressive Web App (PWA) for calculating 529 college savings plans with S&P 500 ETF projections.

## Features
- Calculate monthly contributions needed to reach your goal
- S&P 500 investment projections (7-10% historical returns)
- Progressive Web App - works offline and installable
- Responsive design for all devices

## How to Use
1. Enter your child's date of birth
2. Set your target amount for age 18
3. Add current savings (optional)
4. Choose your expected return rate
5. Get your monthly contribution plan
